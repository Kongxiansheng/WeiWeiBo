# 如何获得ThinkSNS
>* Composer
```shell
composer create-project medz/thinksns
```
>
>* git
```shell
git clone https://github.com/medz/thinksns.git
```
> 克隆下来后，使用composer初始化~
```shell
composer install
```
>* Download
https://github.com/medz/thinksns/releases
下载的zip档案是进过composer初始化后的包，可以直接解压使用，无需使用composer初始化，如果需要升级，请执行
```shell
composer update
```
